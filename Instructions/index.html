Summary: Latitude

The purpose of this exercise was to examine if there was a relationship between Latitude and Weather conditions. Using randomised coordinates, we called the OpenWeatherMap Api to generate data for 500 cities.

There were four weather properties considered. These were: Humidity, Temperature, Cloudiness and Windiness. Pandas was used to conduct the data analysis and MatPlotLib to create the visualisations you see here.
